# About
A Unity game created for class IMG421 3D-Game Engines at NAU to familiarize myself with boid systems in Unity. A simulation of Butterflies flying through a forest.

# Controls
Press Begin to start the simulation.

# Extra Credit
(1) Used Unity's built in Terrain system to add a unique background and floor to the game.

(.5) Reworked the Boid Models to be butterflies by using a free online .obj file I found.

(.5) Added Music: Forest Ambience to the Main Menu and calming music to the simulation.

(1) Added a THEME to the game. The theme is Butterfly migration. Terrain, Boid model, and sound matches this theme accordingly.
